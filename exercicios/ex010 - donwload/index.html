Links

Link Externo

O link externo deve ser usado para carregar outras páginas, exemplo:
Você pode ir para o google clicando aqui

Existe opção "nofollow" que serve para não sincronizar minha página com o link do "unfollow", exemplo:
Patrocinado pelo youtube!

Link Interno

O link interno é usado para mavegação do proprio site, exemplo:

Clique aqui para ir para página a seguir

Link para Donwload

Usando a tag "donwload" e "type apliccation(nome do tipo do arquivo)" dá para fazer o download, exemplo:

  • Clique aqui para baixar

  • Clique aqui para abrir
